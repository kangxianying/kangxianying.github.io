6月 03, 2018 595

UNITTEST介绍

Test Fixture

通过test fixture,可以为一个或多个test进行setup()和teardown().简单地说,setup()是事前执行,teardown()是事后执行

Test Case

是Unittest最小的的组成部分,与assert结合,把检查细化

Test Suit

一起执行多个tests和test case

Test Runner

执行和反馈结果机制

Test Report

汇总多个tests和test case结果

Demo

import unittest

class TestCaseDemo1(unittest.TestCase):
    def setUp(self):
        print("I will run once before every testdemo")

    def test_methodA(self):
        print("Running method A")

    def test_methodB(self):
        print("Running method B")

    def tearDown(self):
        print("I will run after every testdemo")

if __name__ == '__main__':
    unittest.main(verbosity=2)
